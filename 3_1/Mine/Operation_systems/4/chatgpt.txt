Вот пошаговое решение для выполнения вашей лабораторной работы с использованием команд в Linux:

### 1. Вызвать окно терминала или войти в текстовую консоль
- Откройте терминал или войдите в консоль.

### 2. Исполнить команду `yes >/dev/null`
- Введите команду:
   ```bash
   yes >/dev/null
   ```
   Эта команда будет генерировать бесконечный вывод "y", перенаправляя его в "ничто" (в "нулевой" файл).

### 3. Перевести запущенный процесс в фоновый режим
- Чтобы перевести процесс в фоновый режим, используйте комбинацию клавиш:
   ```bash
   Ctrl + Z
   ```
   Это приостановит процесс, после чего нужно выполнить:
   ```bash
   bg
   ```
   Это возобновит процесс в фоновом режиме.

### 4. Запустить процесс `yes abc >/dev/null` сразу в фоновом режиме
- Выполните команду с использованием символа `&` в конце для запуска в фоновом режиме:
   ```bash
   yes abc >/dev/null &
   ```

### 5. Просмотреть список заданий
- Используйте команду:
   ```bash
   jobs
   ```

### 6. Перевести задание номер 2 в режим переднего плана
- Используйте команду `fg` с номером задания:
   ```bash
   fg 2
   ```

### 7. Уничтожить задание на переднем плане
- Уничтожить процесс можно с помощью комбинации клавиш:
   ```bash
   Ctrl + C
   ```

### 8. Для задания 1 определить объём занимаемой им памяти
- Сначала выясните PID процесса:
   ```bash
   jobs -l
   ```
   Затем используйте команду `top` или `ps` для получения информации о памяти:
   ```bash
   ps -o pid,rss -p <PID>
   ```
   Здесь `<PID>` замените на идентификатор процесса.

### 9. Задание номер 1 уничтожить, не переводя его на передний план
- Используйте команду `kill` с номером процесса:
   ```bash
   kill %1
   ```

### 10. Просмотреть список всех процессов, исполняющихся в системе
- Используйте команду:
   ```bash
   ps aux
   ```

### Найти процесс, являющийся родительским для процесса bash, исполняемого на текущем терминале
- Сначала найдите текущий процесс `bash`:
   ```bash
   ps -f
   ```
   В поле `PPID` будет указан идентификатор родительского процесса.
