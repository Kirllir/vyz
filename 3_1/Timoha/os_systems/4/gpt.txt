Для выполнения лабораторной работы по управлению процессами и заданиями в Bash, можно следовать приведенным шагам:

1. **Открыть терминал**:  
   Откройте окно терминала в вашей операционной системе или войдите в текстовую консоль, используя комбинации клавиш (например, `Ctrl + Alt + F1` для текстовой консоли на Linux).

2. **Исполнить команду `yes`**:  
   Выполните команду, которая отправляет бесконечный поток "yes" в `/dev/null`, чтобы создать процесс:
   ```bash
   yes >/dev/null
   ```
   Команда начнет работать в текущем сеансе терминала.

3. **Перевести процесс в фоновый режим**:  
   Для перевода активного процесса в фоновый режим, нужно сначала приостановить его с помощью комбинации клавиш `Ctrl + Z`, а затем выполнить команду для перевода в фон:
   ```bash
   bg
   ```

4. **Запустить процесс `yes abc` сразу в фоновом режиме**:  
   Для запуска команды сразу в фоновом режиме, добавьте амперсанд `&` в конце команды:
   ```bash
   yes abc >/dev/null &
   ```

5. **Просмотреть список заданий**:  
   Чтобы увидеть список всех фоновых заданий, используйте команду:
   ```bash
   jobs
   ```

6. **Определить PID процесса из задания 2**:  
   Вы можете использовать команду `jobs -l`, которая покажет PID процессов для каждого задания:
   ```bash
   jobs -l
   ```
   Найдите процесс, связанный с командой `yes >/dev/null`.

7. **Просмотреть список процессов и определить состояние процесса**:  
   Используйте команду `ps` для отображения всех процессов и информации о них. Например:
   ```bash
   ps -e -o pid,pgid,stat,tty,cmd
   ```
   Здесь `PID` – это идентификатор процесса, `PGID` – номер группы, `STAT` – состояние процесса, а `TTY` – управляющий терминал. Найдите процесс по его PID и посмотрите всю требуемую информацию.

8. **Перевести задание номер 2 в режим переднего плана**:  
   Для этого выполните команду `fg` с номером задания. Например, если задание имеет номер 1:
   ```bash
   fg %1
   ```

9. **Уничтожить задание на переднем плане**:  
   Чтобы завершить процесс на переднем плане, можно использовать комбинацию клавиш `Ctrl + C`, которая отправляет сигнал `SIGINT` для завершения процесса.

10. **Уничтожить задание в фоновом режиме, не переводя его на передний план**:  
    Для этого сначала определите номер задания командой `jobs`, а затем используйте команду `kill`, чтобы завершить процесс. Например, если задание имеет номер 1:
    ```bash
    kill %1
    ```
    Если процесс не завершается сразу, можно отправить более сильный сигнал, например, `SIGKILL`:
    ```bash
    kill -9 %1
    ```

Таким образом, вы сможете выполнить лабораторную работу по управлению процессами и заданиями в Bash.
